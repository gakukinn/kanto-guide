### 2. 地区页面结构 
已发现6个地区，每个都有完整的活动类型页面：

#### 地区列表:
- **tokyo** (东京都)
- **saitama** (埼玉县) 
- **chiba** (千叶县)
- **kanagawa** (神奈川县)
- **kitakanto** (北关东)
- **koshinetsu** (甲信越)

#### 每个地区包含6种活动类型:
- **hanabi** (花火/烟花)
- **matsuri** (祭典)
- **hanami** (花见/赏樱)
- **illumination** (灯光秀)
- **culture** (文艺活动)
- **momiji** (红叶/观枫)

### 3. 活动详情页面
已发现多个具体活动页面，例如：
- `tokyo/hanabi/activity-*` 格式的活动详情页
- `tokyo/matsuri/activity-*` 格式的活动详情页
